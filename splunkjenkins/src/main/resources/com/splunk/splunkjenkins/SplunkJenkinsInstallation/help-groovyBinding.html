Provide a groovy script to customize event process, will be executed after junit report is published

You can sent events to splunk by call: 
    def event=[]
    event["build_url"]=build.url
    event["result"]=testResult
    event["metadat"]=env
    send(event)
you can reference below variables 
 
    AbstractBuild build;
    Map env;
    TestResult testResult;
